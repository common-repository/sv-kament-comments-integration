== Description ==

**SV KAMENT Comments for WordPress**

SV KAMENT is a cloud commenting platform, that makes commenting easy, interactive and feature-rich.

* Threaded comments
* Social integration (FB, Twitter, VK, Odnoklassniki, Mail.RU)
* Real-time
* User rating & levels
* Email notifications of new replies
* Comments indexable by search engines (SEO friendly)
* Powerfull administration & moderation tools
* VK social translations

Full feature description & working demo can be found on our website - [http://svkament.ru](http://svkament.ru)

**------------------------------------------**

**Основные особенности SV KAMENT**

SV KAMENT это платформа комментирования которая позволяет легко превратить ваш сайт в удобную площадку для общения.

* Вложенные комментарии
* Социальная интеграция (Facebook, Twitter, Vkontakte, Odnoklassniki, Mail.RU)
* Трансляция релевантных комментариев из вашей группы Вконтакте и из виджета "Комментарии Вконтакте" на вашем сайте
* Живое общение в реальном времени
* Уведомления о новых ответах на почту
* Рейтинг и уровни пользователей
* Индексация комментариев поисковыми ботами (SEO)
* Удобство администрирования и модерирования

Узнать подробности и попробовать в действии можно на нашем сайте - [http://svkament.ru](http://svkament.ru)

== Installation ==

1. You have to register your new community (website) in SV KAMENT ([Register](http://svkament.ru/create))
2. Upload plugin files to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. In `Plugin Settings` enter you SV KAMENT community shortname (see step 1)

**------------------------------------------**

1. Вам необходимо зарегистрировать на SV KAMENT новое сообщество (сайт) ([Регистрация](http://svkament.ru/create))
2. Загрузите файлы плагина в папку `/wp-content/plugins/` вашего сайта
3. Активируйте плагин в панели администратора
4. В настройках плагина укажите короткое имя вашего сообщества (см. шаг 1)
